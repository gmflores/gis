1. Empezar

1.1. Instalación

En esta sección se presentan los pasos para instalar MATLAB y los complementos necesarios para el tratamiento de datos geoespaciales y la creación de mapas. Se sugiere como organizar las carpetas y como configurar la ruta de acceso a la carpeta de datos.

Nota

Este módulo está dirigido a estudiantes del CICIMAR-IPN y se asume la instalación de MATLAB con licencia institucional (opcionalmente, si no cuentas con una cuenta de correo institucional, puedes utilizar la versión de prueba de MATLAB o MATLAB Online).

1.1.1. MATLAB y complementos

Para instalar MATLAB sigue estos pasos:

  1. Crea una cuenta con tu dirección de correo institucional en el sitio web de MathWorks.

  2. Descarga y ejecuta el instalador de MATLAB.

  3. Durante la instalación, selecciona las Toolboxes que deseas instalar. Asegúrate de incluir las siguientes:

    • Mapping Toolbox

    • Image Processing Toolbox

  4. Complementos opcionales:

    • Statistics and Machine Learning Toolbox

    • Hyperspectral Imaging Toolbox

1.2. Organización de carpetas

Se recomienda utilizar la siguiente estructura de carpetas para organizar los datos y tus scripts de MATLAB:

GIS
├── datos
│   ├── csv
│   ├── kml
│   ├── mat
│   ├── nc
│   ├── shp
│   └── tif
├── matlab
│   ├── gmf_00_tsunamis.mlx
│   ├── gmf_01_sismos.mlx
│ ...

En la carpeta matlab guardarás los scripts (.mlx) que irás realizando durante el curso. Los primeros dos digitos del nombre del archivo corresponden al número consecutivo de la práctica, las siguentes tres letras a tus iniciales y enseguida una o unas palabras que identifique la práctica (por ejemplo: gmf_00_tsunamis.mlx). Los nombres de los archivos deben empezar con una letra, pueden contener letras, números o guiones bajos (sin espacios ni caracteres especiales).

1.3. Establecer la ruta

Para que MATLAB (desde tus scripts) localice los datos, agrega esa carpeta a la ruta de búsqueda desde la pestaña HOME y seleccionando Set Path. Luego selecciona Add with Subfolders... y elige la carpeta datos. Después selecciona Save y Close.
